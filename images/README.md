# Images Folder

Add your images here:

- `logo.png` - Company logo (200x50px recommended)
- `favicon.ico` - Favicon (32x32px)
- `hero-bg.png` - Hero background (1920x1080px)
- `portfolio/` - Portfolio screenshots (800x600px each)
- `team/` - Team member photos (400x400px each)
- `icons/` - Custom SVG icons (64x64px)

## Usage

```html
<img src="images/logo.png" alt="Logo">
<img src="images/portfolio/project-1.jpg" alt="Project">
```
